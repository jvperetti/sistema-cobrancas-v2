# Sistema de Cobranças Nascimento v2

Automação desenvolvida para otimizar o fluxo de cobranças da **Empresa Nascimento**, integrando uma interface moderna com processamento robusto em Python.

## 🚀 Tecnologias Utilizadas
* **Python**: Lógica principal e automação.
* **Eel**: Biblioteca para interface GUI baseada em tecnologias Web (HTML/JS).
* **SQLite**: Banco de dados local para persistência.
* **PyInstaller**: Utilizado para compilação do executável.

## 🛠️ Funcionalidades
* Gestão de cobranças e controle financeiro.
* Interface intuitiva com suporte a assinaturas digitais.
* Banco de dados relacional para histórico de operações.
* Sistema de login e níveis de acesso.
